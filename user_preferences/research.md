사용자의 취향 분석
-> 선호 장르, 가수 사전 수집
-> 규칙이 중심, 취향은 가중치 레이어로 사용 !
-> 규칙대로 리스트를 출력하고 취향으로 재정렬시킴

## 추가변수

pref_genre_bonus: 선호 장르에 보너스 점수를 줌
pref_artist_bonus: 선호 아티스트에 보너스 점수를 줌
seed_mix_strategy: 스포티파이 검색 키워드를 규칙중심으로 할지, 취향중심으로 할지 결정하는 전략 플래그
| 전략                | 설명                         | 예시                                                                      |
| ----------------- | -------------------------- | ----------------------------------------------------------------------- |
| `"context-first"` | 규칙 기반 시드 위주 (취향 거의 반영 안 함) |  카페·focus 규칙 중심으로 `lo-fi`, `chill`, `instrumental` 같은 시드만 넣음          |
| `"hybrid"`        | 규칙과 취향을 섞음 (가장 추천)         |  규칙 시드 3개 + 취향 시드 2개 (예: `lo-fi`, `chill`, `jazz`, `Crush`)           |
| `"pref-first"`    | 취향 위주로 검색 (규칙 최소 반영)       |  `Crush`, `Zion.T`, `jazz`, `R&B` 같은 취향 중심 — relax나 ambient 상황에 주로 사용 |
